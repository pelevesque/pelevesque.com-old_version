Draft | 2D Shape Engine for LÖVE 2D

Draft is a 2D shape drafting engine for the LÖVE 2D game engine.

With draft you can easily draw primitives (lines, triangles, rectangles, polygones), and more luxurious shapes like rhombi, trapezia, kites, parallelograms, etc... A compass function allows drawing of curved shapes like ellipses, semi-circles, and arcs. Linkers allow you to connect shape points using lines in various ways to create spider web type drawings.

Skeletor | 2D Skeletal Animation Engine for LÖVE 2D

Skeletor is a 2D skeletal animation engine for the LÖVE 2D game engine.

It has many features packed into a tight class: 1) A detailed cascading style system to decorate the wireframe, joints, and shapes. 2) A comprehensive tagging system to retain track of skeletons and bones. 3) A skeleton boundary system for collision detection. 4) A clean and comprehensive API. 5) Unit tests for better development and testing.

p.objects | A Collection of Objects for Max/MSP

p.objects is a collection of objects for Max/MSP. They are divided into 3 groups: 1) audio, 2) LCD, 3) utilities.

The audio objects provide various effects like chorus and flanger, as well as fade algorithms and a sampler. The LCD objects are for drawing on the Max/MSP LCD. The utilities help for various small tasks such as creating gates qnd sorting numbers.
